# Station-Manager
GareManagement : Application Java (JDBC/DAO) dédiée à la gestion ferroviaire. Elle centralise le contrôle des gares, trains, billets et personnel. Dotée d'une interface moderne bleue (Tailwind), elle assure un CRUD complet, le suivi des horaires et des ventes en temps réel. Une solution robuste pour optimiser l'administration de réseaux.
GareManagement - Système de Gestion Ferroviaire
🚀 Fonctionnalités

Gestion des Gares : Administration complète des sites (CRUD).

Contrôle du Trafic : Suivi des trains, des destinations et des horaires (Départ/Arrivée).

Billetterie Intégrée : Gestion des types de billets et des tarifs associés.

Gestion du Personnel : Suivi des gestionnaires et de leurs performances de vente.

Historique des Ventes : Journalisation en temps réel de toutes les transactions.

Interface Moderne : Tableau de bord intuitif conçu avec Tailwind CSS (Bleu Thème).

🛠️ Technologies Utilisées

Langage : Java (JDK 11+)

Persistance : JDBC (Java Database Connectivity)

Design Pattern : DAO (Data Access Object)

Base de Données : MySQL / MariaDB

Interface Utilisateur : HTML5, Tailwind CSS

Serveur de Base : Pilotage via JDBC Driver
