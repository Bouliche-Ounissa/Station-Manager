# Station-Manager
GareManagement : Application Java (JDBC/DAO) dédiée à la gestion ferroviaire. Elle centralise le contrôle des gares, trains, billets et personnel. Dotée d'une interface moderne bleue (Tailwind), elle assure un CRUD complet, le suivi des horaires et des ventes en temps réel. Une solution robuste pour optimiser l'administration de réseaux.
